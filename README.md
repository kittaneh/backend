## Description

A NestJs sample code with GraphQL and MongoDB

## Prerequisite

A running local MongoDB server on port 27017

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
